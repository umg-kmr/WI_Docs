PrerequisitesÔÉÅ

  1. Install anaconda by following the intall instructions at anaconda.org for your operating system. It is not essential to the operation of the code suite but recommended to setup a conda environment separately for a WinDynB install.

  2. Create a new conda environment, we will be using windynb as the environment name for illustration.

    conda create -n windynb python

  3. Activate the conda environment created in the previous step and then add essential packages like a suite of compilers (we will use GNU compilers), openmpi.

    conda install -c conda-forge gcc gfortran gxx openmpi

    Note

    You can also use the compilers provided by your system and other compilers like clang although the testing was exclusively done using g++.

  4. Download the boost library from boost.org depending on your operating system and save it to a location of your choice.

    wget <boost-download-link> -P <path-to-save-to>

    There is no installation required for boost library, just extract the downloaded package.

  5. Finally, install Cobaya by following the instructions given in the documentation. Cobaya implements the bayesian analysis part of the code suite.
